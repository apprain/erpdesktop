## Getting Started

### Prerequisites
- Windows OS
- Visual Studio (2019 or later recommended)
- .NET Desktop workload installed

### Run the Project
1. Open `ApprainERPTerminal.sln` in Visual Studio
2. Restore NuGet packages (if prompted)
3. Build the solution
4. Run the project using **Start (F5)**

---

## Build & Deployment

- Build mode: Release
- Installer project is available under `ApprainERPSetup`
- Use Visual Studio installer build to generate deployment package

---

## Version Control Notes

The following folders are excluded via `.gitignore`:
- `bin/`
- `obj/`
- `.vs/`

Sensitive data such as credentials and connection strings should never be committed.
